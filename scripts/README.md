# `scripts`

This folder contains Raycast-friendly shell entrypoints for common actions.

## Scripts

- `index-all.sh`: indexes the default user folders.
- `index-folder.sh`: indexes one user-supplied folder.
- `multimodal-search.sh`: runs CLI search and opens the top result.

## Role In The Project

- These are convenience wrappers around the Python backend in `api/`.
- They are useful for automation and quick local actions without opening the dashboard.
