# `ui-dashboard/raycast-extension`

This folder contains the full Raycast extension implementation for searching and indexing from Raycast.

## Contents

- `package.json`: Raycast extension package definition.
- `package-lock.json`: locked Node dependency tree.
- `tsconfig.json`: TypeScript compiler config.
- `raycast-env.d.ts`: Raycast environment typings.
- `src/`: extension source code.
- `assets/`: static assets for the extension.

## Notes

- `node_modules/` is intentionally excluded from repo documentation because it is dependency output.
- This folder is source code plus JavaScript package metadata for the Raycast integration.
